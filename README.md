# Putevodets-Stromynka

Веб-приложение для построения маршрутов внутри здания учебного корпуса.

## Описание проекта

Система предназначена для поиска и визуализации кратчайшего маршрута между кабинетами здания. Пользователь выбирает начальную и конечную точки маршрута, после чего приложение строит путь и отображает его на плане соответствующего этажа.

Маршрутизация реализована на основе графовой модели здания с использованием алгоритма поиска кратчайшего пути.

## Основные возможности

- построение маршрута между кабинетами;
- поддержка нескольких этажей здания;
- визуализация маршрута на плане этажа;
- веб-интерфейс для взаимодействия с пользователем;
- проверка корректности введённых данных.

## Настройка

Параметры запуска задаются в файле `app.py`:

```python
app.run(host='0.0.0.0', port=8001, debug=False)
```

Доступные параметры:

- `host='0.0.0.0'` — доступ к приложению с других устройств локальной сети;
- `host='127.0.0.1'` — доступ только с локального компьютера;
- `port=8001` — используемый порт;
- `debug=True` — режим отладки для разработки.
